Python Descriptor Notes

Example

class TestDataDescriptor(object):

    def __get__(self,obj,type):
        print 'in __get__'
        return obj.val

    def __set__(self,obj,val):
        print 'in __set__'
        obj.val = val

class Foo(object):
    a = TestDataDescriptor()

    def __init__(self):
        self.a = 1

if __name__=='__main__':

    x = Foo()

    print "reassigning a"
    x.a = 2

Gotchas

Descriptor methods only work for class variables, you can use the object parameter to store data on the instance itself.

The descriptor protocol methods are invoked via an objects __getattribute__ so be careful when overriding
